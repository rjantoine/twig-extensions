# Twig Extensions plugin for Craft CMS 3.x

Adds the extra filters and functions from [Twig Extensions](http://twig-extensions.readthedocs.io/en/latest/)

## Requirements

This plugin requires Craft CMS 3.0.0-beta.23 or later.

## Installation

To install the plugin, follow these instructions.

1. Open your terminal and go to your Craft project:

        cd /path/to/project

2. Then tell Composer to load the plugin:

        composer require rjantoine/twig-extensions

3. In the Control Panel, go to Settings → Plugins and click the “Install” button for Twig Extensions.

## Using Twig Extensions

Read the [documentation](http://twig-extensions.readthedocs.io/) for more information.
